Тестовое задание.
Приложение для управления пользователями.
===========================================================
База данных содержится в папке DB в файле test.sql.
===========================================================
Каталоги:

Корневой каталог

	Файлы:

	index.php 
	Осуществляет контроль данных, выводимых в браузер.
	Весь диалог с приложеним происходят через него.
	
Каталог CSS
Хранит все файлы стилей (*.css) приложения.

	Файлы:
	
	style.css
	Осуществляет центрирование некоторых элементов.
	
Каталог js
Хранит все файлы с разрешением *.js.

	Файлы:
	
	ajax.js
	Организация асинхронного вывода информации.
	
	jquery-3.3.1.min.js
	Файлы библиотеки jquery-3.
	
Каталог view
Содержит все файлы, отвечающие за вывод информации в окно браузера.

	Файлы:
	
	editor.php
	Страница редактирования данных пользователей.
	
	footer.php
	Вывод части статических элементов веб-страницы.
	
	header.php
	Вывод части статических элементов веб-страницы.
	Подключение библиотеки jquery-3.
	Подключение стилей.
	
	login.php
	Форма аутентификации пользователей.
	
	menu.php
	Вывод условного меню приложения.
	Отображение Логина пользователя и его роли(статуса: admin, user).
	
	users.php
	Вывод всех возможных пользователей, возможная сортировка по полю "name", "dateCreated".
	Для пользователя с ролью "admin" вывод ссылок для редактирования и удаления пользователей.
	
Каталог script
Содержит файлы, отвечающие за работу с БД, обработку данных и вывод данных в формы из каталога view.

	Файлы:
	
	const.php
	Объявляет константы.
	
	connection.php
	Осуществляет подключение к БД.
	
	auth.php
	Аутентификация пользователя.
	
	deleteUser.php
	Удаление пользователя.
	
	edit.php
	Редактирование пользователя.
	
	logout.php
	Закртыие сессии, выход пользователя из системы.
	
	userInfo.php
	Вывод информации по конкретному пользователю с возможностью редактирования.
	
	viewUsers.php
	Вывод некоторой информации(ФИО, Login, Дата создания) по пользователях.
	Сортировка пользователей.
	Пагинация.
